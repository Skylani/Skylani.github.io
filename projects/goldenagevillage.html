While volunteering to lecture elders living in Golden Age Village, a senior apartment, about usage of smart devices, I had the opportunity to create a website for this nice living community.

View Website

This was my first time building a website with WordPress by customizing a theme. Through reading blog posts, watching YouTube videos and most importantly making this website, I've learned a lot.

Tools

  • CMS: WordPress
  • Theme: Esteem Free Edition
  • WP Plugin: Contact Form 7 - Contact form.
  • WP Plugin: Breadcrumb NavXT - Adds a breadcrumb navigation showing the visitor's path to their current location.
  • WP Plugin: Foo Gallery - Gallery management tool.
  • WP Plugin: Polylang - Adds multilingual capability.
  • WP Plugin: Shortcodes Ultimate - Pack of shortcodes.
  • WP Plugin: Black Studio TinyMCE Widget - Adds a new "Visual Editor" widget type based on the native WordPress TinyMCE editor.
  • WP Plugin: Yoast SEO - An all-in-one SEO solution.
  • WP Plugin: Google XML Sitemaps - Generate an XML sitemap.
  • WP Plugin: WP Super Cache - Very fast caching plugin.
