The Odin Project

The Odin Project is a website for learning web development knowledge which includes HTML5, CSS3, Ruby, Ruby on Rails, JavaScript and jQuery. It stitches together all kinds of online resources into a curriculum so that I can follow a learning path instead of blindly studying anything I can find on the internet.

In this curriculum, I get to build dozens of projects from simple scripts to full websites/programs. Here you can see some front-end projects.

Note-taking app I use for studying web development

When working on the project, I used Evernote to take note but found it not suitable for studying a particular topic because of its notebook structure. Then, I changed my note-taking app to OneNote mainly because that it can better organize notes into sections and pages. Although these two are great note-taking apps, I was not satisfied. I also wanted Markdown support.

In the meantime, I came across an idea that learn by writing a book. Therefore, when I started working on the HTML5 and CSS3 section of the curriculum, I used GitBook as a note-taking app. This way, I can better organize my newly found knowledge. Better yet, it supports Markdown.
